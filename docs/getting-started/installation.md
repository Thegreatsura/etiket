# Installation

## Package Manager

```sh
# npm
npm install etiket

# pnpm
pnpm add etiket

# yarn
yarn add etiket

# bun
bun add etiket
```

## Requirements

- Node.js 18+ (or any modern runtime with ESM support)
- TypeScript 5+ (optional, for type checking)

## ESM Only

etiket is published as pure ESM. Make sure your project uses `"type": "module"` in `package.json` or use `.mjs` file extension.

```json
{
  "type": "module"
}
```
